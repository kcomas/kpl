# KPL (Work In Progress)

## Typed/Jitted Scripting Language Written In C

### Supported Platforms

#### x64 Linux

### Dependencies

#### GNU Make
#### GCC

### Build

```bash
make
```
##### Debug Build Flags (DFLAGS)

ALD=1 - Track allocations and frees

### Run

```bash
./kpl path/to/file
```
##### Examples are in the ex folder
