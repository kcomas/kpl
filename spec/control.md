
# Control Flow
## If

### If
if (cond) { body }

### If with Else If
if (cond) { body } ei (cond) { body }

### If with else
if (cond) { body } ee { body }

### If with else if and else
if (cond) { body } ei (cond) { body } ee { body }

## While
we (cond) { body }

## Do While
do { body } we (cond)

## For
fr (var;cond;per) { body }
