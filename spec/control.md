
# Control Flow

## If

### If
? (cond) { body }

### If with Else If
? (cond) { body } (else if cond) { else if body }

### If with else

? (cond) { body } { else body }

### If with else if and else .....
? (cond) { body } (else if cond) { else if body } { else body }

## While

?? (cond) { body }
