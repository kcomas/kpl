
# Operators

## Variables

### : Define/Assign

Create and or update a vars value

```
var : statement

var`symbol : statement

var(accessor) : statement
```

### :: Shadow

Create a var with the same name as one in the parent scope

### Casting $

```
var : TYPE $ data
```

### Swapping <:>

Swap the contents of two variables

```
var <:> var
```
