
# Regex

PCRE Regular Expressions

## Using

String ~ "mode/pattern/subsitition/flags"\regex

String ~ Regex ( mode\`mode; "pattern"\`pattern; "subsitition"\`subsitition; flags\`flags )

## Flags

See .regex.Flags

### d

Add indices to matches

### g

Global

### i

Case insensitive

### m

Make ^ $ match line start and end

### s

. matches newline
