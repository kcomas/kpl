
# Regex

PCRE Regular Expressions

## Creating

```
Regex { flags; "subsitition"; "pattern" }

~ "pattern"

flags ~ "pattern"

flags ~ "subsitition" ~ "pattern"
```

## Flags

See .regex.Flags

### d

Add indices to matches

### g

Global

### i

Case insensitive

### m

Make ^ $ match line start and end

### s

. matches newline
